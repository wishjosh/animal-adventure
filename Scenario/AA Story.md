## 📌 설계 원칙

```
목표:     12마리 수호대를 모아 초록별을 구한다 (목적지향)
방법:     블록을 직접 쌓고 파서 서식지를 만든다 (샌드박스)
학습:     행동하기 전에 도감 카드로 이유를 먼저 안다
감정:     동물과의 관계가 쌓이며 자연에 대한 다정함이 자란다
```

## 🎬 프롤로그: 잠든 행성의 부름

낡은 나무 상자 속에서 발견한 회색빛 지도. 여섯 개의 땅 중 오직 **초록 마을**에만 희미한 불빛이 깜빡입니다.

지도 뒷면에 한 줄:

> *"초록별이 숨 쉬는 법을 잊어가고 있어. 땅과 물과 바람을 다스릴 12마리의 수호대를 찾아 행성의 심장을 다시 뛰게 해 줘."*
> 

**반복 의식 — 매 레벨 시작마다:**
수호대 멤버들이 한 명씩 등장하며 외친다.

> *"탐험하고, 구조하고, 지켜라!"*
>

## 🔁 영입 메커니즘 4유형

모든 동물이 같은 방식으로 오지 않는다.

| 유형 | 방식 | 적용 동물 |
| --- | --- | --- |
| **A. 환경 조성형** | 올바른 서식지 블록을 만들면 스스로 찾아옴 | 꿀벌, 양, 수달, 두루미 |
| **B. 구조형** | 다치거나 갇힌 동물을 발견·구조·이송 | 제비, 박쥐, 너구리 |
| **C. 신뢰 쌓기형** | 여러 번 만나며 조금씩 가까워짐 | 붉은여우, 반달가슴곰 |
| **D. 문제 해결형** | 동물을 위협하는 원인을 찾아 제거 | 연어, 독수리, 황조롱이 |

## 🌟 3단계 영입 시스템

모든 수호대는 한번에 합류하지 않는다.

```
1단계 — 발견 (Encounter)
  동물의 흔적·소리 확인
  → 도감 카드 1차 (특징·서식지)

2단계 — 도움 (Assist)
  한 번 도와줌. 하지만 동물은 떠남.
  → 도감 카드 2차 (위협 요인)

3단계 — 합류 (Bond)
  동물이 스스로 다시 찾아옴. 영구 합류.
  → 도감 카드 3차 (수호대 능력)
  → 합류 연출 + BGM
```

## 📖 자연 도감 카드 시스템

영입 직전 짧은 카드가 뜬다. 서진이가 카드를 읽고 스스로 "그러면 이렇게 해야겠구나"를 추론하는 구조.

```
┌─────────────────────────────┐
│  🐝 토종 꿀벌               │
│─────────────────────────────│
│  사는 곳: 꽃이 있는 곳       │
│  특기: 하루 수백 송이 수분   │
│  위협: 농약, 꽃가루 없는 꽃  │
│──────────────────────────── │
│  💡 클로버·해바라기를        │
│     심어주면 좋아해요        │
└─────────────────────────────│
         클릭하면 닫힙니다
```

## 🟡 LEVEL 1: 초록 마을 — "메마른 땅의 첫 숨결"

**테마:** 생산자와 분해자 | 생명과 돌봄
**배경:** 딱딱하게 굳은 흙, 시들어가는 고목나무

### 교란자: 길 잃은 멧돼지 가족

텃밭을 자꾸 헤집는 멧돼지 가족. 쫓아내는 게 아니라 그들의 이동 경로를 파악해서 **우회로 블록**을 만들어줘야 한다. 첫 등장부터 "공존"의 감각을 심는다.

### 일회성 구조 동물

- **다친 참새 "짹짹이"** — 고목나무 아래 떨어진 새끼. 안아서 나뭇가지 위에 올려주면 엄마가 데려감. 서진이의 첫 구조 경험.

### 미션 흐름

```
Phase 0 — 단서 탐색
  노란 구슬 4개 클릭
  → 도감 카드 (고목나무·지렁이·꿀벌·오리)
  → 할머니 등장: "모든 게 연결되어 있단다"

Phase 1 — 독성 식물 제거
  ⚠️ 독성 식물 → 삽으로 제거
  → 말이 풀밭으로 돌아옴

Phase 2 — 토마토 텃밭 복원
  젖은 흙 만들기 → 씨앗 심기 → 물 주기
  → 바질 곁에 심으면 무당벌레 등장 → 진딧물 퇴치
  → 도감 카드: 동반식물·공생 관계

Phase 3 — 지렁이 부활
  낙엽 5장 수집 → 고목나무 근처 흙에 덮기
  → 지렁이 미니게임 (16칸 클릭)
  → 영양토 생성 → 고목나무 회복 시작

Phase 4 — 고목나무 개화
  영양분이 뿌리로 스며드는 파티클
  → 나무에 잎이 돋고 열매 맺음
  → 그늘 블록 생성
```

### 수호대 합류 (영입 유형별)

**1. 토종 꿀벌 🐝 — 환경 조성형 (A)**
도감 카드: "꽃가루 있는 꽃을 심어야 와요"
→ 클로버·해바라기 블록 3개 이상 심기
→ 꿀벌이 날아와 주위를 맴돌다 합류

**2. 강남 간 제비 🦅 — 구조형 (B)**
도감 카드: "진흙으로 둥지를 짓고 처마 밑을 좋아해요"
→ 강가에 진흙 웅덩이 파기
→ 다친 새끼 제비 발견 → 안아서 처마 위로 이송
→ 어미 제비가 돌아와 합류

**3. 점박이 양 🐑 — 환경 조성형 (A)**
도감 카드: "더위에 약해요. 그늘과 볏짚을 좋아해요"
→ 고목나무 복원으로 그늘 블록 생성
→ 볏짚 깔기 → 다친 양 이송 → 치료
→ 건강해진 양이 합류

### 레벨 1 클리어

```
수호대 3마리 합류 완료
→ 지도 안개 걷힘: "다양성의 숲" 발견
→ 도감 완성: 고목나무·지렁이·꿀벌·제비·양·말·
            염소·오리·무당벌레·토마토·바질
→ 레벨 2 잠금 해제
```

## 🟢 LEVEL 2: 다양성의 숲 — "잃어버린 주소들"

**테마:** 서식지와 다양성
**배경:** 세 생태계가 맞닿은 숲. 동물들이 원래 자리를 잃고 방황.

### 교란자: 황소개구리 무리

토종 도롱뇽을 잡아먹는 외래종. 다 없애는 게 아니라 **격리 연못 블록**을 만들어 옮기는 방식으로 해결. 과학 포인트: "외래종은 천적이 없어서 번식력이 압도적이에요."

### 일회성 구조 동물

- **길 잃은 어린 두꺼비 "두꾸"** — 습지가 아닌 바위 지대에서 발견. 안아서 습지로 이송.
- **빛 노출로 혼란스러운 새끼 박쥐 "밤이"** — 동굴 입구에 쓰러진 채 발견.

### 수호대 합류

**4. 천연기념물 수달 🦦 — 환경 조성형 (A)**
도감 카드: "깊은 물이 있어야 살 수 있어요. 오염에 매우 약해요."
→ 막힌 흙 블록 파내어 깊은 물길 열기
→ 미끄럼틀 타듯 물길로 내려오며 합류

**5. 황금박쥐 🦇 — 구조형 (B)**
도감 카드: "1cm 빛만 들어와도 잠을 못 자요. 한국에 100마리도 안 남았어요."
→ 무너진 동굴 천장을 돌 블록으로 빈틈없이 막기
→ 빛이 차단되자 박쥐가 깊은 잠에서 깨어나 합류

### 핵심 학습: 서식지 추리

```
도감 카드 단서 → 서식지 조건 추론 → 블록으로 구현

"밤에 활동, 초음파, 거꾸로 매달림"
→ 어두운 동굴 천장 필요 → 돌 블록으로 밀봉

"촉촉한 피부, 알을 물에 낳음"
→ 습지 경계부 필요 → 물길 블록 연결
```

### 보스 챌린지: 서식지 복원 완성도

외래종 황소개구리·붉은귀거북 격리 연못 완성도 측정. 토착종이 제자리를 찾을수록 외래종이 자연스럽게 밀려난다.

### 레벨 2 클리어

```
수호대 2마리 합류 (누적 5마리)
→ 도감 완성: 수달·박쥐·왜가리·반딧불이·도롱뇽·
            두꺼비·오색딱따구리·황소개구리
→ "연결의 평원" 잠금 해제
```

## 🔵 LEVEL 3: 연결의 평원 — "보이지 않는 고리"

**테마:** 먹이사슬과 공생
**배경:** 풀이 다 뜯어 먹혀 사막이 된 광활한 초원

### 교란자: 떠돌이 들개 무리

포식자가 사라진 초원에 들개가 들어와 생태계 균형을 흔든다. 쫓아내지 말고 **서식 구역 블록**으로 영역을 분리하는 방식으로 해결.

### 일회성 구조 동물

- **다리를 다친 어린 노루 "초롱이"** — 도로 위에 쓰러져 있음. 안아서 안전한 풀밭으로.

### 수호대 합류

**6. 소백산 붉은여우 🦊 — 신뢰 쌓기형 (C)**
도감 카드: "경계심이 매우 강해요. 천천히 다가가야 해요."

```
1회차: 먹이 블록 놓기 → 멀리서 쳐다보다 사라짐
2회차: 다시 먹이 놓기 → 조금 가까이 옴
3회차: 가만히 앉아서 기다리기 → 코를 킁킁 대다 합류
```

과학 포인트: 관목 블록으로 시야 차단 "완충 지대"를 지어주자 돌아와 초원의 폭발적인 번식을 조절.

**7. 독수리 🦅 — 문제 해결형 (D)**
도감 카드: "썩은 동물을 먹어 전염병을 막아요. 하늘의 청소부예요."
→ 들판에 방치된 사체 블록 발견 → 처리
→ 평원의 생명 순환이 회복되자 날아와 합류

### 핵심 장면: 늑대 없는 세상 시뮬레이션

붉은여우 합류 전, 짧은 타임랩스 연출.

```
"풀이 너무 많아졌어요" (1계절 후)
"초식동물이 너무 늘었어요" (1년 후)
"목초지가 사막이 되었어요" (3년 후)
→ "그래서 포식자가 필요해요"
```

### 보스 챌린지: 먹이사슬 퍼즐 타워

```
생산자 → 1차소비자 → 2차소비자 → 최상위포식자

블록 제거 → 연쇄 붕괴 시작
제한시간 안에 올바른 블록으로 채우기
독수리 호출 → 탑 전체 조망으로 빈자리 확인 가능
```

### 레벨 3 클리어

```
수호대 2마리 합류 (누적 7마리)
→ 도감 완성: 붉은여우·독수리·노루·들개·
            메뚜기·들꽃·황새
→ "강의 근원지" 잠금 해제
```

## 🟣 LEVEL 4: 강의 근원지 — "산과 바다의 약속"

**테마:** 물과 숲의 순환
**배경:** 산 정상 수원지부터 하구까지

### 교란자: 상류 무허가 펜션 주인

강을 오염시키는 펜션 주인. 첫 사람 캐릭터 등장. 쫓아내는 게 아니라 **대화와 협상**으로 오염 저감 블록 설치에 동의시키는 방식. "사람도 먹고 살아야 하는 현실"을 다룬다.

### 일회성 구조 동물

- **강바닥에 갇힌 어린 쏘가리 "쏘야"** — 쓰레기 댐에 막혀 이동 못함. 블록 제거로 물길 열기.

### 수호대 합류

**8. 두루미 🕊️ — 환경 조성형 (A)**
도감 카드: "맑은 습지와 얕은 물가를 좋아해요. 기후 변화에 매우 민감해요."
→ 산비탈에 나무 블록 빽빽이 심어 녹색 댐 건설
→ 정화 습지 조성 → 두루미가 고고하게 내려앉아 합류

**9. 남대천 연어 🐟 — 문제 해결형 (D)**
도감 카드: "태어난 강으로 돌아오는 본능이 있어요. 물이 막히면 죽어요."
→ 펜션 주인 설득 → 오염 저감 블록 설치
→ 강바닥 시멘트 보(洑) 발견·제거 → 물길 연결
→ 연어가 거친 물살 거슬러 올라와 합류

### 핵심 미션: 녹색 댐

```
나무 없음 → 폭우 시 즉시 범람 시뮬레이션
나무 있음 → 폭우 흡수 후 서서히 방류
           → 뿌리 아래 파란 물방울 파티클
           → 잎에서 수증기 구름 형성
```

### 보스 챌린지: 100년 홍수

```
60초 전 폭우 경고
복원한 숲 양만큼 방어력 결정

완벽 복원 → 피해 없음
절반 복원 → 마을 일부 침수
복원 안 함 → 레벨 5 난이도 상승
수달 호출 → 취약 수중 구간 실시간 알림
두루미 호출 → 폭우 방향·강도 사전 감지
```

### 레벨 4 클리어

```
수호대 2마리 합류 (누적 9마리)
→ 도감 완성: 두루미·연어·쏘가리·수달(활용)·
            왜가리·버드나무·갈대
→ "경계 도시" 잠금 해제
```

## 🔴 LEVEL 5: 경계 도시 — "회색 섬의 징검다리"

**테마:** 개발과 자연의 균형
**배경:** 도시와 자연이 맞닿은 경계 지대

### 교란자: 시청 개발 담당 공무원

8차선 도로 확장 계획자. 도로를 막는 게 아니라 **생태 통로 설계 대안**을 제시해서 승인을 받는 방식. "나쁜 사람이 아니라 몰랐던 사람"으로 묘사.

### 일회성 구조 동물

- **쓰레기에 발이 묶인 너구리 새끼 "라쿤이"** — 도심 하천변 발견. 쓰레기 블록 제거로 구출.

### 수호대 합류

**10. 도심 너구리 🦝 — 구조형 (B)**
도감 카드: "실제로 서울·성남 도심 하천변에 살아요! 야행성이라 밤에 활동해요."
→ 8차선 도로 위에 흙+나무 블록으로 생태 육교 건설
→ 너구리가 안전하게 하천으로 건너가며 합류

**11. 황조롱이 🦅 — 문제 해결형 (D)**
도감 카드: "도심 빌딩에 둥지를 틀어요. 도시 쥐를 잡아 먹어요."
→ 삭막한 빌딩 옥상에 흙 블록+텃밭(비오톱) 건설
→ 건물 사이 생태 통로 점검
→ 황조롱이가 옥상 텃밭에 둥지 틀며 합류

### 생태 통로 설계 조건

```
도로 위: 육교형 (초목 블록)
도로 아래: 터널형 (야행성용, 조명 없음)
하천 복원: 물길 따라 동물 이동
통로 폭: 최소 2칸 + 양끝 은신용 관목 블록
```

### 보스 챌린지: 도시 설계 심사

```
완성 후 3가지 실전 테스트:
  테스트 1: 폭우 → 생태 통로 배수 기여도
  테스트 2: 야간 → 동물들 실제 통로 이용 여부
  테스트 3: 시민 반응 → 공원·녹지 활용도
```

### 레벨 5 클리어

```
수호대 2마리 합류 (누적 11마리)
→ 도감 완성: 너구리·황조롱이·도심참새·
            담쟁이덩굴·옥상정원식물
→ "초록별 심장부" 잠금 해제
→ 비밀 해금: 레벨 1~5 생태계가 하나의 네트워크로 연결
```

## ⚪ LEVEL 6: 초록별 심장부 — "모든 것은 연결되어 있다"

**테마:** 기후와 지구 시스템
**배경:** 전 지구 규모

### 브릿지 장면

```
레벨 1 초록 마을의 파란 물방울 하나가
강을 타고 → 바다로 → 대기를 타고 → 북극까지
흘러가는 10초 파티클 연출

두루미: "저 파란 방울이 보여?
        우리 마을에서 출발한 거야.
        작은 것 하나가 지구 전체와 연결되어 있어."
```

### 교란자: 기후 변화 자체

레벨 1~5의 모든 교란자(멧돼지·황소개구리·들개·펜션·공무원)가 사실 하나의 근원과 연결되어 있었음이 밝혀진다.

### 최종 탐험: 기후 이상 진원지

```
세계 지도 이상 신호 5곳:

신호 1 — 북극: 빙하 급속 용해
  → 두루미 파견 (레벨 4 합류 필수)

신호 2 — 아마존: 대규모 산불
  → 독수리 파견 (레벨 3 합류 필수)

신호 3 — 산호초: 백화 현상
  → 수달+연어 파견 (레벨 2·4 합류 필수)
  ※ 연어는 냉수성 → 산호초 아닌 북태평양 수계 담당

신호 4 — 시베리아: 영구동토 해빙
  → 두루미+연어 수계 파견

신호 5 — 초록 마을: 탄소 흡수량 급감
  → 양 파견 (레벨 1 합류 필수)
```

### 수호대 합류: 반달가슴곰 — 신뢰 쌓기형 (C)

**12. 지리산 반달가슴곰 🐻 — 신뢰 쌓기형 (C)**
도감 카드: "지리산에 복원 중이에요. 경계심이 매우 강하고 도토리를 좋아해요."

```
11마리가 모였지만 심장은 여전히 미약.
지금까지 쌓아온 모든 생태계 연결이
도토리 숲 한가운데서 빛을 발하면
깊은 숲속에서 반달가슴곰이 처음으로 모습을 드러냄.

1회차: 도토리 블록 놓기 → 발자국만 남기고 사라짐
2회차: 더 많은 도토리 → 멀리서 바라봄
3회차: 움직이지 않고 기다리기 → 천천히 다가와 합류
```

### 피드백 루프 미션

```

```

