## 2026-06-06

### 레벨 3 들개 격리 및 모바일 HUD 수정

**문제**
- 들개 무리를 관목/울타리로 격리해도 미션이 통과되지 않는 문제가 있었다.
- 들개가 관목이나 울타리 밖으로 나가려 할 때 몸 일부가 블록과 겹쳐 보였다.
- 모바일 화면에서 방위표 아래 목적지 표시 박스가 우측 조작 버튼과 겹쳤다.

**수정**
- `js/world.js`에 레벨 3 격리벽 판정 헬퍼를 추가해 지면 위에 설치된 관목, 울타리, 돌, 흙 등을 정확히 감지하도록 했다.
- `js/Level3Logic.js`의 들개 격리 BFS가 기존처럼 벽 위쪽 빈 칸을 검사하지 않고 실제 설치된 격리벽을 검사하도록 변경했다.
- 들개 이동 시 중심점만 검사하던 방식을 몸 반경 샘플 검사로 바꿔 관목/울타리에 파고들기 전에 방향을 틀도록 했다.
- 모바일/작은 화면에서는 방위표와 목적지 박스를 왼쪽 상단으로 배치해 우측 `빠름/점프/하강` 버튼과 겹치지 않게 했다.

### 레벨 3 먹이사슬 퍼즐 모바일 드래그 수정

**문제**
- 레벨 3 후반 먹이사슬 퍼즐 타워에서 카드를 슬롯으로 드래그할 수 없었다.
- 기존 구현은 HTML5 `dragstart/drop` 이벤트에 의존했는데, iPhone/iPad 터치 환경에서는 이 방식이 안정적으로 동작하지 않는다.

**수정**
- `js/Level3Logic.js`의 먹이사슬 퍼즐 카드 배치 로직을 `placeCardInSlot()`로 분리했다.
- 기존 마우스 HTML5 드래그를 유지하면서, 모바일용 `PointerEvent` 기반 드래그를 추가했다.
- 작은 화면에서 더 쉽게 조작할 수 있도록 `카드 탭 -> 슬롯 탭` 배치 방식도 추가했다.
- 배치한 카드는 PC에서는 더블클릭, 모바일에서는 탭으로 다시 제거할 수 있게 했다.

**검증**
- `Get-ChildItem -Recurse -Filter *.js | ForEach-Object { node --check $_.FullName }`
- `git diff --check`
